# FAQ / トラブルシューティング

> このドキュメントは育てる前提のテンプレートです。よくある質問が出てきたら追記します。

## CSS が反映されない

確認するポイント:

1. **ポップアップでトグルが ON か** — ホストごとに有効/無効を切り替えます。
2. **ホスト名が一致しているか** — 設定はホスト名単位（例 `www.example.com`）です。サブドメインが違うと別設定になります。
3. **サイト側のスタイルに負けていないか** — CSS に `!important` を付けるか、セレクタの詳細度を上げてください。
4. **拡張をリロードしたか** — dist を再ビルドした後は `chrome://extensions` で拡張の更新（リロード）を押してください。

content script が注入されない技術的背景は [ADR-0001](./adr/0001-content-script-bundling.md) を参照。

## 設定を別の PC へ移したい

オプションページの **エクスポート** で JSON を保存し、移行先で **インポート**してください。インポートは全置換のため、移行先の既存設定は上書きされます。

## 設定はどこに保存される？

`chrome.storage.local`（端末ローカル）です。詳細は [データモデル](./data-model.md) と [プライバシーポリシー](./privacy.md) を参照。
