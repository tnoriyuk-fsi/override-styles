# プライバシーポリシー

> このドキュメントは整備中です（issue #14）。Chrome Web Store 申請の前提として正式版を用意します。

## 収集する情報

Override Styles は、ユーザーの個人情報を収集・外部送信しません。

- ユーザーが入力した CSS と各ホストの有効/無効設定のみを、`chrome.storage.local`（端末ローカル）に保存します。
- 外部サーバへの送信、解析ツール、トラッキングは行いません。

## データの保存場所

- すべての設定は利用者のブラウザ内（`chrome.storage.local`）に保存されます。
- import/export はユーザーが明示的に操作したときのみ、ローカルファイルとして読み書きされます。

## 第三者提供

- 収集するデータがないため、第三者への提供はありません。
