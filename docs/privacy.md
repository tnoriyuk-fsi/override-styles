# プライバシーポリシー

最終更新日: 2026-06-18

Override Styles（以下「本拡張機能」）のプライバシーに関する方針を説明します。Chrome Web Store の審査・掲載で参照できる正式版です。

## 収集する情報

**本拡張機能は、利用者の個人情報を一切収集しません（no data collection）。**

- 取り扱うのは、利用者自身が入力した CSS と、各ホストの有効/無効設定のみです。
- 外部サーバへの送信、アクセス解析、トラッキング、広告は一切行いません。
- 閲覧履歴・Cookie・フォーム入力などのページ内容を読み取ることはありません。

## データの保存場所

- すべての設定は利用者のブラウザ内（`chrome.storage.local`）にのみ保存されます。端末の外へ送信されることはありません。
- import/export は、利用者が明示的に操作したときに限り、ローカルファイルとして読み書きされます。

## 権限とその理由

本拡張機能が要求する権限は、機能の実現に必要な最小限です。

| 権限                           | 用途                                             | 理由                                                      |
| ------------------------------ | ------------------------------------------------ | --------------------------------------------------------- |
| `storage`                      | ホストごとの CSS 設定を保存する                  | 機能の中核。`chrome.storage.local` に端末ローカル保存     |
| `tabs`                         | ポップアップでアクティブタブのホスト名を取得する | 現在表示中サイトの設定を編集するため                      |
| `host_permissions: <all_urls>` | 任意のサイトに CSS を適用する                    | 利用者が設定した任意ホストで CSS 注入を行う必要があるため |

> これらの権限は、利用者が指定したサイトに CSS を適用し、その設定を端末内に保存する目的にのみ使用します。データの外部送信には使用しません。

## 第三者提供

- 収集・保存するデータを外部に送信しないため、第三者への提供はありません。
- 解析サービスや広告ネットワークなど、第三者のサービスも利用しません。

## ポリシーの変更

本ポリシーを変更した場合は、本ページを更新し、冒頭の「最終更新日」を改めます。
